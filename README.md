# Varotra Boutique - Fullstack Application

Une application de boutique moderne construite avec React, Tailwind CSS, Express et Supabase.

## Structure
- `client/` : Frontend React (Vite, TypeScript, Tailwind CSS)
- `server/` : Backend Express (TypeScript, Node.js)

## Fonctionnalités
- Page d'accueil avec section Hero attractive
- Grille de produits responsive avec filtres
- Système de "Commande" (Simulation)
- Intégration Supabase (Prête pour la production)

## Installation

1. Installez les dépendances à la racine :
   ```bash
   npm run install-all
   ```

2. Configurez vos variables d'environnement dans `client/.env.local` et `server/.env`.

3. Lancez l'application (client et serveur simultanément) :
   ```bash
   npm start
   ```

## Technologies
- **Frontend** : React 19, Vite, Tailwind CSS 4, Lucide React
- **Backend** : Express, TypeScript, Supabase JS
- **Base de données** : Supabase (PostgreSQL)
